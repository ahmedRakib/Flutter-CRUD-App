## Running the backend
You need to have the following installed:
1. [Java JDK 8+](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html)
2. [Maven](https://maven.apache.org/download.cgi)

Make sure you have java and maven installed by running

- ```java -version```
- ```mvn -version```

Finally, inside the **frontend-for-cross-usage-backend** folder run:

``` mvn spring-boot:run ```
